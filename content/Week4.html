一、Onshape介紹

Onshape是一種計算機輔助設計（CAD）軟件系統，通過軟件即服務（SAAS）模型在Internet上提供。它廣泛使用雲計算，在基於Internet的服務器上執行計算密集型處理和渲染，用戶可以通過Web瀏覽器或iOS和Android應用程序與系統進行交互。

參考資料 :  https://en.wikipedia.org/wiki/Onshape

二、Onshape課程內容

  1. 建立 Onshape 帳號 (學校gmail)
  2. 簡單介紹其介面操作
  3. 使用 Onshpae 繪製及練習簡單圖形
  4. 講解草圖建模步驟
  5. 做擠出等等它項指令
  6. 完成簡單圖形

三、Onshape簡單說明

快捷鍵F為零件擺回中間的位子

sketch (草圖)

feature (特徵)

part studio (零件)

Assembly (組立)

Drawing (工程圖)

folder (目錄)

四、ONSHAPE基本用法:

1. Extrude (擠出)

2. revolve (旋轉)

3. sweep (掃掠)

4. loft (斷面混成)

5. thicken (加厚)

6. shell (薄殼)

7. pattern (陣列)

8. split (切割)
