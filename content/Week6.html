二、更新flaskapp.py 至最新版本

  1. Google 搜尋 cmsimfly
  2. git clone cmsimfly 的倉儲 ( git clone https://github.com/chiamingyen/cmsimfly  )
  3. 將 cmsimfly 裡的 flaskapp.py 複製貼上至自己的資料夾，更新 flaskapp.py
  4. 在 Leo 中，開啟 cmsimfly.leo
  5. 在 Leo 中的編輯 flaskapp.py 下按 ctrl + i 新增一個 @auto flaskapp.py
  6. 鼠標在 @auto flaskapp.py 上，按右鍵點 Refresh from disk
  7. 將原本的 @clean flaskapp.py 刪掉
  8. 把 @auto flaskapp.py 改為 @clean flaskapp.py
