一、git 指令

git init (建立一個.git資料)

git log (查看版次)

git branch 分支名 (創建新分支)

git checkout 分支名 (轉換分支)

git log --oneline --graph --all (檢查各提交間的關聯)

git merge 分支名 (合併分支)

二、什麼是計算機輔助設計？

計算機輔助設計（CAD）是使用計算機系統（或工作站）來幫助創建，修改，分析或優化設計。CAD軟件用於提高設計人員的工作效率，提高設計質量，通過文檔改進通信，以及創建製造數據庫。CAD輸出通常採用電子文件的形式，用於打印，加工或其他製造操作。還使用術語CADD（用於計算機輔助設計和繪圖）。

Computer Aided (輔助)

Guided (導引)

Automated (自動)

Autonomous (自主) Design

參考資料 : https://en.wikipedia.org/wiki/Computer-aided_design
